# Contributing to Advent of AI Challenge Repository

Thank you for your interest! While this repository primarily documents my personal journey through Block's Advent of AI challenge, contributions are welcome in certain forms.

## Ways You Can Contribute

### 1. Reporting Issues

If you find any issues with the projects:
- Check if the issue already exists
- Create a new issue with a clear description
- Include steps to reproduce (if applicable)
- Mention which Day/Project the issue relates to

### 2. Suggesting Improvements

Have ideas for improving existing projects?
- Open an issue with the "enhancement" label
- Describe your suggestion clearly
- Explain why it would be beneficial

### 3. Sharing Your Own Advent of AI Journey

Participating in the challenge yourself?
- Share your repository link in discussions
- Create your own Advent of AI repository
- Use this repo as inspiration (with attribution)

### 4. Documentation Improvements

Help improve documentation:
- Fix typos or unclear instructions
- Add missing setup steps
- Improve code comments
- Update outdated information

## How to Submit Changes

1. **Fork the repository**
2. **Create a branch**: `git checkout -b fix/improvement-name`
3. **Make your changes**
4. **Test thoroughly**
5. **Commit**: `git commit -m "Description of changes"`
6. **Push**: `git push origin fix/improvement-name`
7. **Open a Pull Request**

## Pull Request Guidelines

- **Keep changes focused** - One PR per issue/feature
- **Test your changes** - Ensure everything works
- **Follow existing code style** - Match the project's conventions
- **Update documentation** - If you change functionality
- **Be respectful** - Follow the code of conduct

## Code of Conduct

- Be respectful and constructive
- Welcome newcomers
- Focus on the project, not personalities
- No harassment or discrimination
- Help create a positive learning environment

## Questions?

- Open an issue with the "question" label
- Check existing issues and documentation first
- Be specific about what you need help with

## Learning Resources

If you're doing the Advent of AI challenge yourself:
- [Goose Documentation](https://github.com/block/goose)

## Note About Challenge Projects

Since this repository documents a personal learning journey, major feature additions to existing projects may not be merged. However, bug fixes, documentation improvements, and suggestions are always welcome!

## Attribution

When using code from this repository:
- Provide attribution to the original project
- Link back to this repository
- Follow the MIT License terms
- Credit Block's Advent of AI challenge
